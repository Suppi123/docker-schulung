Ersetze die ? mit den passenden Elementen.

## Wichtige Docker Compose Befehle

### Container bauen
`docker compose build`

### Container starten
`docker compose up`

Die Container im Hintergrund starten:

`docker compose up -d`

### Die Container stoppen
Wenn die Container im Vordergrund laufen:
`STRG + C`

Wenn die Container im Hintergrund laufen:

`docker compose stop`

Wenn die komplette Compose-Umgebung abgebaut werden soll:

`docker compose down`
