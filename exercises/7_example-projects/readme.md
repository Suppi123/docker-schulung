# Bonusaufgaben - Praxisbeispiele
In dieser Aufgabe versuchen wir eine Anwendung aus der Praxis mithilfe von Docker Compose auf unserem System zu starten.

## Projektvorschläge
### Portainer
#### Beschreibung
Web-UI zum Managen von Docker-Containern und Stacks.
#### Herausforderungen
Einfacher Einstieg in Compose-Management, visuell zugänglich.

### Uptime Kuma
#### Beschreibung
Monitoring-Tool für Websites und Services.
#### Herausforderungen
Zeigt Ports, Volumes und Restart-Policies; schnell testbar.

### Homepage
#### Beschreibung
Einfaches Dashboard für Selfhosted-Services.
#### Herausforderungen
Minimales YAML, ideal zum Anpassen und Erweitern.

### Vaultwarden
#### Beschreibung
Password-Manager (Bitwarden-kompatibel).
#### Herausforderungen
Auth-Demo mit DB-Volume, skalierbar für Multi-Service

### Home Assistant
#### Beschreibung
Home-Automation-Hub
#### Herausforderungen
Komplexeres Setup mit Add-ons, zeigt Netzwerke.
